# Library Management System - SQL Analysis

## Project Overview
This project analyzes library operations using MySQL.

## Objectives
- Track issued books
- Analyze member activity
- Monitor book availability
- Generate business insights

## Tools Used
- MySQL
- SQL
- GitHub

## Key SQL Concepts
- Joins
- Aggregate Functions
- Group By
- Subqueries
- Window Functions

## Sample Insights
- Most borrowed books
- Active members
- Monthly issue trends
- Fine collection analysis
